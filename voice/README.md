## Goal

Give Jarvis an actual voice for audio briefings, updates, and responses.

## Options

### 1. ElevenLabs (Premium)
- Natural, high-quality voices
- Multiple voice options (British butler, Iron Man JARVIS-style)
- API-based
- Cost: ~$0.30 per 1,000 characters

### 2. OpenAI TTS (Good balance)
- Built into OpenAI API (already have key)
- 6 preset voices (Alloy, Echo, Fable, Onyx, Nova, Shimmer)
- Cost: $0.015 per 1,000 characters
- Good quality, natural sounding

### 3. macOS `say` (Free, local)
- Built into macOS
- Multiple voices available
- No API cost
- Lower quality but functional

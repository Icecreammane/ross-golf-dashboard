## 08:25 - "Lean" Goal Calculator - SHIPPED ‚úÖ

### Feature Request
Ross: "We should add to the calorie fitness tracker 'lean' a feature where you enter goals and it recommends target calories based on what you are trying to achieve"

### What Got Built
**Science-based goal calculator using Mifflin-St Jeor BMR formula**

**Inputs:**
- Current weight
- Goal weight
- Timeline (weeks)
- Activity level (sedentary ‚Üí very active)
- Age
- Height

**Calculations:**
1. **BMR** (Basal Metabolic Rate) - Base calories burned at rest
2. **TDEE** (Total Daily Energy Expenditure) - BMR √ó activity multiplier
3. **Required deficit/surplus** - (goal - current) √ó 3500 cal/lb
4. **Daily target** - TDEE ¬± daily deficit
5. **Protein recommendation** - 1g/lb for cutting, 0.8g/lb for maintenance

**Safety Checks:**
- Warns if rate >2 lbs/week (aggressive)
- Floors calories at 1500 minimum
- Shows weekly rate projection

**UI:**
- Floating "üéØ Set Goals" button (bottom left)
- Modal calculator with form
- Real-time calculation
- Shows: recommended calories, protein, TDEE, daily deficit, weekly rate
- One-tap "Save & Apply" updates dashboard targets

**API Endpoints:**
- POST /api/calculate_goals - Returns recommendations
- POST /api/save_goals - Saves to user_goals.json + updates dashboard

**Files:**
- fitness-tracker/app_pro.py (added 2 endpoints)
- fitness-tracker/templates/lean_calculator.html (new modal)
- fitness-tracker/templates/dashboard_v3.html (integrated)

**Live:** http://10.0.0.18:3000 ‚Üí Click "üéØ Set Goals" (bottom left)

**Example Output:**
- Current: 225 lbs ‚Üí Goal: 190 lbs (26 weeks)
- Moderate activity (3-5 days/week)
- Result: 2,458 cal/day, 225g protein, -1.35 lbs/week

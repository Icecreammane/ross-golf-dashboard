Data Flow:
----------

1. Manual Entry (daily):
   bash finance_entry.sh → enter data → saves to JSON

2. Automated Run (6am daily):
   launchd triggers → financial_tracker.py → creates snapshot → logs to file

3. View Reports (anytime):
   bash finance_report.sh → reads JSON → calculates metrics → displays report

Metrics Calculated:
-------------------

From snapshots → calculations:
- Daily average expenses
- Weekly projection (daily × 7)
- Monthly run rate (daily × 30)
- Savings rate: (revenue - expenses) / revenue
- Runway: balance / monthly expenses
- Expense breakdown by category
- Goal projections (90-day rolling average)


Goal Tracking:
--------------

Florida Fund:
  Target: $50,000
  Formula: (target - current) / monthly_net = months_to_goal

Financial Independence:
  Target: $900,000 (4% rule for $3K/month)
  Formula: (900K - savings) / monthly_net = months_to_FI
